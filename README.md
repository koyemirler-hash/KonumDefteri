# Konum Defteri PWA

## Kurulum

### 1. İkonları Yap
make-icons.html dosyasını tarayıcıda aç → "192 indir" ve "512 indir" linklerine tıkla → icon-192.png ve icon-512.png olarak kaydet → bu klasöre koy

### 2. GitHub'a Yükle
- GitHub Desktop uygulamasıyla ya da git push ile yükle

### 3. GitHub Pages Aç
Settings → Pages → Branch: main → / (root) → Save

### 4. Kullan
https://koyemirler-hash.github.io/KonumDefteri/

### Güncelleme
Her push'ta GitHub Actions otomatik deploy eder.
Uygulama yeni versiyonu arka planda indirir, sayfa yenilenince güncellenir.
